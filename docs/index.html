Getting Started

Supercell API Client is a comprehensive JavaScript library for interacting with official Supercell game APIs: Clash of Clans, Clash Royale, and Brawl Stars.

Full API Coverage

All endpoints for CoC, CR and BS with 50+ methods

Rate Limiting

Built-in throttling and auto-retry with exponential backoff

TypeScript Support

Full type definitions included out of the box

Proxy Support

Built-in proxies for development without static IP

Installation

npm install @vladnet14/supercell-api

Usage

import SupercellClient from '@vladnet14/supercell-api';

const client = new SupercellClient({
  token: 'YOUR_API_TOKEN',
  apiType: 'clashofclans', // or 'clashroyale', 'brawlstars'
  useProxy: false, // set true if you don't have a static IP
});

// Get player
const player = await client.endpoints.getPlayer('#PLAYER_TAG');
console.log(`Player: ${player.name}, Level: ${player.expLevel}`);

// Get clan
const clan = await client.endpoints.getClan('#CLAN_TAG');
console.log(`Clan: ${clan.name}, Level: ${clan.clanLevel}`);

// Search clans
const clans = await client.endpoints.searchClans({ name: 'CRUVO', limit: 10 });
console.log('Found clans:', clans.items?.map((c) => c.name));

API Reference

Clash of Clans

getPlayer(playerTag)

Fetch player information by tag.

Parameters:
  • playerTag (string) — Player tag with or without '#'

verifyPlayerToken(playerTag, token)

Verify player API token.

Parameters:
  • playerTag (string) — Player tag
  • token (string) — Token to verify

searchClans(params)

Search for clans using filters.

Parameters:
  • params.name (string) — Clan name
  • params.warFrequency (string) — War frequency
  • params.minMembers (number) — Minimum members
  • params.maxMembers (number) — Maximum members
  • params.minClanPoints (number) — Minimum clan points
  • params.minClanLevel (number) — Minimum clan level
  • params.limit (number) — Max results (1-50)
  • params.after (string) — Pagination cursor
  • params.before (string) — Pagination cursor
  • params.labelIds (string) — Label IDs comma separated
